## What it does

Minecraft Utilities is a web-based toolkit focused on planning, tracking, and reference work for Minecraft. Instead of jumping between separate apps, you get a single place for organizing builds, watching material needs, and looking up gameplay details.

Inside the suite are tools for resource tracking, inventory organization, enchantment planning, brewing references, and seed map viewing. A progress dashboard helps keep ongoing work visible, and JSON export/import support makes it easy to move your setup between sessions or keep a saved backup of your data.

## Included Tools

- Build planner for shaping projects before blocks are placed
- Resource tracker for keeping an eye on materials needed for tasks and constructions
- Enchantment calculator for comparing item and enchantment combinations
- Brewing guide for checking potion recipes and required inputs
- Seed map viewer for browsing world seed details in a browser
- Inventory organizer for sorting and managing stored items
- Progress dashboard for a fast overview of planning and collection status
- Export and import support through JSON for saving or moving your data

## Compatibility

Minecraft Utilities is built for Web delivery and ships as HTML-based content. It is intended for modern desktop browsers and can rely on standard browser features being available.

Known limitations:
- It is a browser utility, not an in-game mod
- Some views or saved data may depend on local browser storage
- Display behavior can vary across browsers and screen sizes

## FAQ

**How do I get started?**  
Download the build, open the HTML file, and choose the tool you want to use first.

**Can I transfer my settings or saved data?**  
Yes. The JSON export and import features let you keep a copy or move your data elsewhere.

**Does it update every tool separately?**  
The suite is built as a shared toolkit, so a release can affect several planning and reference features together.

**Can I adapt the workflow to my needs?**  
Yes. You can decide which panels and reference tools matter most and rely on those more heavily.

**Is it tied to a single Minecraft version?**  
No specific client version is required; since the project is browser-based, compatibility depends more on your browser and how you use the utility.

**Where is the best place to keep the files?**  
Any folder you can open locally or serve from the web will work, but a dedicated project folder makes exports and assets easier to manage.
